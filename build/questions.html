Instructions

Memory

Simply press the red circle to start. Certain shapes will "light up" in a certain order. Observe the order. Once it's finished, repeat the order. If you do it correctly, you earn one point, and the pattern becomes one longer. Otherwise, the game simply reloads and you go back to zero. This game is most similar to the "Simon Game", which is a physical, handheld device with the same concept. "

Attention

In every round, there are 99 sad faces and 1 happy face. Your task is to spot the happy face. Once you do so, record the row and column in the inputs provided. If you do it correctly, the happy face will move to a different location and your inputs would clear.

Reason

A jumbled word will appear on the screen. Your task is to unscramble the word. Once you do so, type the word in the input provided. If you do it correctly, the word will change. Otherwise, nothing will change. Note that there may be multiple words that can be formed from the jumbled word.

Function

A simple math problem will appear on the screen. Type the correct answer in the input provided. If you do so, the problem will change. Otherwise, nothing will change.
